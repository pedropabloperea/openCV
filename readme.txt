SPANISH

Una vez que has importado el código cpp en tu entorno de trabajo, tienes que importar la carpeta "multimedia" a la raíz de dicho entorno de trabajo.

Después deberás situar los archivos dll en la carpeta donde se generarán o colocarás los archivos ejecutables del proyecto.


ENGLISH

Once you have imported the cpp code to your workspace, you have to import the "multimedia" folder into the root of that workspace.

Then you should place the provided dll files into the folder where the executable files of the project will be created.
